> if の基本構文を確認しましょう。   
>> if 条件式  
　条件式が真の場合に実行する処理  
end  

```ifの後ろには条件式を書きます。条件式の結果が「真」である場合は、ifの中の処理が実行されます。```  
```言い換えれば、条件式の結果が「偽」である場合は、処理が行われません。```  

  
ifの条件を満たすルールは "条件がfalseまたはnilは条件を満たさず、それ以外は条件を満たす"  
  
今回のプログラムで書かれた条件 age < 20 や true は、falseでもnilでもないので、ifの条件を満たされます。
